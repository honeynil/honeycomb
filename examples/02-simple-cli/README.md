# Пример 2: Simple CLI Calculator

## Что делает этот пример?

CLI-калькулятор с базовыми математическими операциями (сложение, вычитание, умножение, деление).

- Несколько логических компонентов
- Код можно разделить по чёткой ответственности
- Не требуется приватность (нет переиспользования в других пакетах)

## Структура

```
02-simple-cli/
├── main.go       # CLI интерфейс, роутинг команд
├── calc.go       # Бизнес-логика вычислений
├── format.go     # Форматирование и вывод
├── go.mod        # Зависимости
└── README.md     # Документация
```

## Почему именно так?

- **Все файлы в одном пакете `main`** — Не создаём лишние пакеты преждевременно.
- **Файлы названы по функциональности** — `calc.go` содержит логику вычислений, `format.go` — форматирование вывода. Это читаемо и понятно.
- **Нет директорий `pkg/`, `internal/`, `cmd/`** — они не нужны на этом этапе. Не усложняй преждевременно.

### Разделение по ответственности
- `main.go` → знает о CLI аргументах
- `calc.go` → знает о математике
- `format.go` → знает о красивом выводе

## Когда переходить дальше?

- Появился код, который нужно скрыть (внутренние детали)
- Нужно переиспользовать код в тестах
- Работает команда разработчиков
